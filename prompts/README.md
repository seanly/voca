# Voca Prompts 集合

各种用于 Voca 语音输入应用的 Prompt 模板，可直接复制到 Voca 设置中使用。

## 语音纠错类

| 文件 | 说明 |
|------|------|
| [speech-recognition-correction.md](speech-recognition-correction.md) | 通用语音识别纠错，修正中英文转录错误 |
| [programming-speech-correction.md](programming-speech-correction.md) | 编程场景语音纠错，涵盖 Git、Docker、编程语言等术语 |

## 文本处理类

| 文件 | 说明 |
|------|------|
| [text-polishing.md](text-polishing.md) | 文本润色，提升语法和可读性 |
| [chinese-english-translation.md](chinese-english-translation.md) | 中英文双向翻译 |
| [code-formatting.md](code-formatting.md) | 代码格式化和风格统一 |

## 开发工具类

| 文件 | 说明 |
|------|------|
| [git-commit-message.md](git-commit-message.md) | 口述变更 → Conventional Commit 格式的提交信息 |
| [code-comment.md](code-comment.md) | 口述功能 → JSDoc/Javadoc/docstring 等代码注释 |
| [sql-query.md](sql-query.md) | 口述需求 → SQL 查询语句 |
| [shell-command.md](shell-command.md) | 口述操作 → 可执行的 Shell 命令 |

## 工作协作类

| 文件 | 说明 |
|------|------|
| [meeting-notes.md](meeting-notes.md) | 语音转录 → 结构化会议纪要 |
| [tech-proposal.md](tech-proposal.md) | 口述想法 → 结构化技术方案文档 |
| [bug-report.md](bug-report.md) | 口述问题 → 标准 Bug 报告格式 |

## 使用方法

1. 选择适合您场景的 Prompt 文件
2. 复制文件中的 Prompt 内容
3. 在 Voca 应用设置中新建 Prompt，粘贴内容即可
